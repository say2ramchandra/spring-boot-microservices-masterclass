ℹ️ About This Demo

This application demonstrates Spring Security 6 concepts for the Spring Boot Microservices Masterclass.

🎯 What You'll Learn

  • Form-Based Authentication - Custom login page with username/password
  • HTTP Basic Authentication - REST API protection
  • Role-Based Access Control (RBAC) - USER, MANAGER, ADMIN roles
  • URL-Based Security - Using authorizeHttpRequests()
  • Method-Level Security - Using @PreAuthorize and @Secured
  • Remember-Me - Persistent authentication
  • Custom UserDetailsService - Database-backed authentication
  • Password Encoding - BCrypt hashing

🏗️ Architecture

  • Spring Boot 3.2 - Application framework
  • Spring Security 6 - Security framework
  • Spring Data JPA - Database access
  • H2 Database - In-memory database (dev mode)
  • Thymeleaf - Server-side templating with security extras

📚 Key Files

  • SecurityConfig.java - Security filter chains configuration
  • CustomUserDetailsService.java - Loads users from database
  • User.java / Role.java - JPA entities
  • DataInitializer.java - Seeds demo users
  • WebController.java - Thymeleaf page controllers
  • ApiController.java - REST API endpoints
